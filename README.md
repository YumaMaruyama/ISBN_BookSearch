# SpringBoot_BookSearch

## [アプリ概要]
本に記載されてある13桁のINBNコードを入力すると本の詳細が確認できます。  
入力フォームにISBNコードを入力し検索すると、本の表紙や帯をみてもわからないような情報が確認できます。  
* アプリURL： http://133.167.92.108:8083/isbnSearchHome
